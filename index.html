Agglayer Helm Charts

Production-ready Kubernetes applications for the Agglayer ecosystem

Quick Start

Add the Agglayer Helm repository and start deploying:

# Add the Agglayer Helm repository
helm repo add agglayer https://agglayer.github.io/helm-charts

# Update your local Helm chart repository cache
helm repo update

# Install a chart (example with aggkit)
helm install my-aggkit agglayer/aggkit

Available Charts

aggkit

CDK

Aggregation kit for Polygon CDK - handles transaction aggregation and batch processing.

agglayer

CDK

Aggregation layer service that provides unified settlement across multiple chains.

blockscout

Explorer

Open-source blockchain explorer for viewing transactions, blocks, and smart contracts.

bridge

Interoperability

Cross-chain bridge service for secure asset transfers between networks.

bridge-ui

Frontend

Web interface for the cross-chain bridge, providing user-friendly asset transfers.

cdk-erigon

Node

High-performance Ethereum client optimized for Polygon CDK chains.

op-geth

Optimism

Optimism's fork of go-ethereum, optimized for Layer 2 operations.

op-node

Optimism

Optimism node implementation for running Layer 2 consensus.

permissionless-nodes

Infrastructure

Deploy and manage permissionless validator nodes across multiple networks.

pool-manager

Management

Manages transaction pools and mempool operations for optimal performance.

op-succinct

ZK

Succinct zero-knowledge proof system components for Optimism rollups.
